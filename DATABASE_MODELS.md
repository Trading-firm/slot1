## 1. `trades` Collection
Stores details for every trade (open and closed).

| Field | Type | Description |
|---|---|---|
| `_id` | ObjectId | Auto-generated by MongoDB |
| `symbol` | string | Market name (e.g., "Volatility 25 Index") |
| `direction` | string | "BUY" or "SELL" |
| `entry_price` | float | Price at which the trade was opened |
| `sl` | float | Stop Loss price |
| `tp` | float | Take Profit price |
| `lot_size` | float | Volume of the trade |
| `ticket` | int | Unique MetaTrader 5 ticket ID |
| `strategy` | string | Name of the strategy used (e.g., "supertrend") |
| `timeframe` | string | Chart timeframe (e.g., "M5", "H1") |
| `rsi_at_entry` | float | RSI indicator value at the time of entry |
| `atr_at_entry` | float | ATR indicator value at the time of entry |
| `ema_trend` | float | EMA indicator value at the time of entry |
| `status` | string | "OPEN" or "CLOSED" |
| `exit_price` | float \| null | Closing price (null if open) |
| `exit_reason` | string \| null | "TP", "SL", or "MANUAL" (null if open) |
| `pnl` | float \| null | Profit or loss in USD (null if open) |
| `pnl_pct` | float \| null | PnL as a percentage of the account balance |
| `entry_time` | datetime | Timestamp when the trade was opened |
| `exit_time` | datetime \| null | Timestamp when the trade was closed |

## 2. `daily_summary` Collection
One record per day with performance stats.

| Field | Type | Description |
|---|---|---|
| `_id` | ObjectId | Auto-generated |
| `date` | string | YYYY-MM-DD (unique index) |
| `total_trades` | int | Number of trades closed that day |
| `winning_trades`| int | Number of profitable trades |
| `losing_trades` | int | Number of unprofitable trades |
| `win_rate` | float | Percentage of wins (0.0 to 100.0) |
| `total_pnl` | float | Total daily profit/loss in USD |
| `best_trade` | float | Highest profit from a single trade |
| `worst_trade` | float | Lowest loss from a single trade |
| `open_trades` | int | Number of trades still open at summary time |
| `updated_at` | datetime | Last update timestamp |

---

## 3. `bot_state` Collection
Persistent key-value storage for the bot.

| Field | Type | Description |
|---|---|---|
| `_id` | ObjectId | Auto-generated |
| `key` | string | Unique key (e.g., "balance", "last_cycle") |
| `value` | any | Value associated with the key |
| `updated_at` | datetime | Last update timestamp |

---

## 4. `market_performance` Collection
Cumulative performance stats per individual market.

| Field | Type | Description |
|---|---|---|
| `_id` | ObjectId | Auto-generated |
| `symbol` | string | Market name (unique index) |
| `total_trades` | int | Lifetime trades for this market |
| `wins` | int | Lifetime wins |
| `losses` | int | Lifetime losses |
| `total_pnl` | float | Lifetime profit/loss |
| `created_at` | datetime | When this market was first traded |
| `updated_at` | datetime | Last trade update |
